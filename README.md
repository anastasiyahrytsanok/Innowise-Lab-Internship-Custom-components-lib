# How to run the app:
Install the library:

npm install @anastasiyahrytsanok/custom-components-lib

Use in your project:

import { Button } from "@anastasiyahrytsanok/custom-components-lib";

function App() {
  return <Button label="Click me" />;
}

Run locally:

npm install
npm start

Run Storybook:

npm run storybook

Build the library:

npm run build
